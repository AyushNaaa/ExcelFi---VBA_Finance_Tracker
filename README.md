# ExcelFi---VBA_Finance_Tracker
A fully functioning, automated, user-oriented finance tracker with Investment Projections, Budgeting, Automated Advice, Visuals, and more. Built entirely on Excel VBA with the ability of storing thousands of unique data points/sets in local macros efficiently.

• Designed and programmed a personal finance management system using Excel VBA to support structured
income/expense tracking, financial summaries, and future planning
• Implemented validated data-entry workflows via custom UserForms, enforcing numeric, categorical, and date
constraints, storing 1000+ unique datasets locally within Excel macros
• Developed automated financial analytics to aggregate transactions, compute net cash flow, and dynamically
updating 10+ visual dashboards
• Built investment growth projections with adjustable multi-year assumptions and interactive graphs

**All projections on investments are just that, projections, nothing is guaranteed in the market, the pipeline uses global averages
